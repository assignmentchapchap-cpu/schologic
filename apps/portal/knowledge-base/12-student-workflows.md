# Student Workflows (Instructor Reference)

## Overview

This document explains how students interact with Schologic LMS. Understanding student workflows helps you:
- Explain the platform to students
- Troubleshoot student issues
- Anticipate student questions
- Design better assignments

---

## How Students Join a Class

### Student Steps

1. **Log In to Student Account**
   - Navigate to Schologic login
   - Enter student credentials

2. **Find Join Class Option**
   - On dashboard, find **Join Class** button

3. **Enter Invite Code**
   - Type the 6-digit code you provided
   - Example: "ABC123"

4. **Confirm Enrollment**
   - System validates code
   - Student confirms joining

5. **Class Appears on Dashboard**
   - New class shows in their class list
   - Can access immediately

### Instructor Notification

- You receive notification of new enrollment
- Check class roster to verify

### Common Issues

| Issue | Solution |
|-------|----------|
| "Invalid code" | Verify code is correct, class not locked |
| Can't find button | Dashboard > Join Class |
| Already enrolled | Code only works once per student |

## How Students Submit Assignments

### Text Submission

1. **Open the Assignment**
   - From class page or dashboard
   - Click on assignment title

2. **View Assignment Details**
   - Read instructions
   - Note due date and requirements

3. **Enter Text**
   - Type directly in text editor
   - Or paste from external source

4. **Preview (Optional)**
   - Some interfaces show AI preview
   - Not final score, just estimate

5. **Click Submit**
   - Confirm submission
   - Cannot be undone (unless allowed)

6. **View Confirmation**
   - Submission timestamp recorded
   - AI analysis runs automatically

7. **See Results**
   - AI score displayed
   - Segment breakdown available

### File Upload Submission

1. **Open the Assignment**
   - Navigate to assignment page

2. **Click Upload File**
   - Find file upload button
   - File picker opens

3. **Select File**
   - Choose DOCX or TXT file
   - Must be allowed format

4. **Wait for Processing**
   - File uploaded
   - Text extracted automatically

5. **Confirm and Submit**
   - Review extracted content
   - Click Submit

6. **View Results**
   - Same as text submission
   - AI analysis complete

### What Happens Behind the Scenes

```
Student submits text/file
    ↓
System records submission
    ↓
Text extracted (if file)
    ↓
AI detection runs automatically
    ↓
Global score calculated
    ↓
Segment breakdown generated
    ↓
Results displayed to student
    ↓
Instructor notified
```

## How Students Take Quizzes

### Quiz Flow

1. **Open Quiz Assignment**
   - Navigate to the quiz
   - Click to start

2. **Answer Questions**
   - Read each question
   - Select answer (A, B, C, D)
   - Move to next question

3. **Complete All Questions**
   - Answer every question
   - Review if desired

4. **Submit Quiz**
   - Click Submit
   - Cannot change answers after

5. **View Instant Results**
   - Score calculated immediately
   - See correct/incorrect answers
   - Total points displayed

### Quiz Features for Students

- All questions visible at once
- Can skip and return
- Submit when ready
- Immediate feedback

## What Students See After Submission

### Submission View

| Element | Description |
|---------|-------------|
| **Submission Status** | "Submitted" with timestamp |
| **AI Score** | Global AI probability percentage |
| **Segment Breakdown** | Color-coded paragraph analysis |
| **Grade** | Displayed after instructor grades |
| **Feedback** | Written comments (if provided) |

### AI Score Visibility

Students see:
- Their AI score immediately
- Segment-by-segment breakdown
- Color coding (green/yellow/red)
- Overall risk indicator

### Grade Visibility

- Visible once instructor saves grade
- Shows points earned
- Rubric breakdown (if available)

## Accessing Class Resources

### Student Steps

1. **Open Class**
   - From dashboard, click class

2. **Go to Resources Section**
   - Find resources area in class view

3. **View Available Materials**
   - Documents, cartridges, links listed
   - Each with type indicator

4. **Open Resource**
   - Click to open
   - Universal Reader launches

5. **Study Content**
   - Navigate cartridge TOC
   - Read documents
   - Use AI summary if available

## Helping Students

### Common Student Questions

**"Why is my AI score high?"**
- Explain AI detection is probabilistic
- Review their writing process
- Discuss acceptable AI use

**"I can't submit my assignment"**
- Check due date hasn't passed
- Verify file format is allowed
- Ensure class isn't locked

**"Where do I find my grade?"**
- Grades appear after instructor posts
- Check submission page
- Look in class Grades section

**"How do I join a class?"**
- Dashboard > Join Class > Enter code
- Provide correct 6-digit code

### Directing Students

If students have technical issues:
1. Have them describe the exact problem
2. Ask for error messages
3. Check if issue is account-related
4. Verify they're using correct role (student vs instructor)
