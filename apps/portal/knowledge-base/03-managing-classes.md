# Managing Classes

## Creating a New Class

### Steps to Create a Class

1. **Navigate to Classes**
   - Click **Classes** in the sidebar menu

2. **Click Create Class**
   - Look for the **+ Create Class** or **New Class** button
   - A form modal will appear

3. **Enter Class Details**
   - **Class Name**: Full name of your course (e.g., "Introduction to Psychology")
   - **Class Code**: Short identifier (e.g., "PSY101")
   - **Start Date**: When the class begins (optional)
   - **End Date**: When the class ends (optional)

4. **Click Create**
   - System generates a unique 6-digit **Invite Code** automatically
   - Your new class appears in the class list

## Inviting Students

### Steps to Share the Invite Code

1. **Open Your Class**
   - Click on the class card from the Classes page

2. **Find the Invite Code**
   - The 6-digit code is prominently displayed (e.g., "ABC123")
   - You can copy it with one click

3. **Share with Students**
   - Include in your syllabus
   - Announce in class
   - Send via email or LMS announcement
   - Post on course website

4. **Monitor Enrollments**
   - View enrolled students in the class overview
   - Receive notifications when students join

### What Students Do
1. Log into their student account
2. Click **Join Class** on their dashboard
3. Enter the 6-digit invite code
4. Confirm enrollment

## Managing Class Resources

### Steps to Add Resources

1. **Open Your Class**
   - Navigate to the class you want to update

2. **Go to Resources Tab**
   - Click the **Resources** tab

3. **Click Add Resource**
   - Choose your source:

4. **From Library**
   - Browse your existing assets
   - Select documents or cartridges
   - Click **Add Selected**

5. **Upload New**
   - Click **Upload** button
   - Select file from your computer
   - Supported: PDF, DOCX, IMSCC
   - Add title if needed

6. **Add Link**
   - Enter external URL
   - Add a descriptive title
   - Useful for videos, websites

### What Students See
- Resources appear in their class view
- Can open documents in Universal Reader
- Download links available

## Configuring Class AI Settings

### Steps to Configure AI Detection

1. **Open Class Settings**
   - Go to your class
   - Click **Settings** tab or gear icon

2. **Enable Custom Settings**
   - Toggle **Override Global Defaults** if you want class-specific settings
   - Otherwise, your profile defaults apply

3. **Select AI Model**
   | Model | Best For |
   |-------|----------|
   | RoBERTa Large | General academic writing |
   | PirateXX Detector | ChatGPT-style content |
   | OpenAI RoBERTa Base | Alternative detection |

4. **Choose Granularity**
   | Setting | Use When |
   |---------|----------|
   | Paragraph | Essays, long-form content |
   | Sentence | Short responses, quizzes |

5. **Select Scoring Method**
   | Method | Description |
   |--------|-------------|
   | Weighted | Nuanced scoring (recommended) |
   | Strict | High-confidence only (>90%) |
   | Binary | Any flagged segment counts |

6. **Save Settings**
   - Click **Save Settings**
   - Changes apply to new submissions

## Locking a Class

### Purpose
Prevent new submissions after a semester ends or during exam periods.

### Steps to Lock

1. **Open Class Settings**
   - Navigate to your class
   - Go to Settings tab

2. **Toggle Lock**
   - Find **Lock Class** toggle
   - Switch to ON

3. **Confirm**
   - Students can no longer submit assignments
   - Existing submissions remain accessible
   - Unlock anytime by toggling OFF

---

## Exporting Grades

### Steps to Export

1. **Open Your Class**
   - Navigate to the class

2. **Go to Grades Tab**
   - Click **Grades** in the class navigation

3. **Click Export to PDF**
   - Button typically in top-right area
   - PDF downloads automatically

### What's Included
- Student names and registration numbers
- All assignment grades
- Total percentage
- Average AI scores
- Class summary statistics

## Class Settings Reference

| Setting | Description | Default |
|---------|-------------|---------|
| AI Model | Detection algorithm | From profile |
| Granularity | Paragraph or Sentence | Paragraph |
| Scoring Method | Weighted/Strict/Binary | Weighted |
| Late Submissions | Allow after due date | Enabled |
| Allowed File Types | TXT, DOCX | Both |
| Lock Status | Prevent submissions | Unlocked |
