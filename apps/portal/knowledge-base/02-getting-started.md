# Getting Started with Schologic LMS

## First Login

### Steps to Log In

1. **Navigate to the login page**
   - Open your web browser
   - Go to your Schologic LMS URL

2. **Enter your credentials**
   - Type your registered email address
   - Enter your password

3. **Click Sign In**
   - You'll be redirected to your Instructor Dashboard

4. **Explore the Dashboard**
   - View your classes at a glance
   - See recent submissions
   - Check AI detection metrics

---

## Setting Up Your Profile

### Steps to Configure Your Profile

1. **Access Profile Settings**
   - Click your avatar or name in the top navigation bar
   - Select **Profile** or **Settings** from the dropdown

2. **Update Personal Information**
   - **Name**: Your display name visible to students
   - **Email**: Your contact email
   - **Avatar**: Upload a profile picture (optional)

3. **Configure Global AI Defaults**
   - Set your preferred AI detection model
   - Choose default granularity (Paragraph/Sentence)
   - Select default scoring method (Weighted/Strict/Binary)
   - These settings apply to all new classes you create

4. **Save Your Changes**
   - Click **Save** to apply your settings
   - Settings take effect immediately

## Navigation Overview

### Main Sidebar Menu

| Menu Item | Purpose | When to Use |
|-----------|---------|-------------|
| **Dashboard** | Overview of all classes, submissions, AI metrics | Daily check-in, quick overview |
| **Classes** | Create and manage your courses | Course administration |
| **Library** | Store and organize documents and cartridges | Resource management |
| **Lab** | Test AI detection settings in a sandbox | Before configuring classes |
| **Calendar** | View assignment due dates across all classes | Planning, deadline tracking |
| **Settings** | Account and global preferences | Initial setup, preference changes |

### Top Navigation

| Element | Function |
|---------|----------|
| **Logo** | Click to return to Dashboard |
| **Notifications Bell** | View new submissions, enrollments |
| **Profile Avatar** | Access profile, settings, logout |

## Understanding the Dashboard

### Dashboard Sections

**Class Overview Cards**
- Quick stats for each class
- Number of students enrolled
- Pending submissions to grade
- Average AI scores

**Recent Submissions**
- Latest student submissions across all classes
- Quick access to grading
- AI score indicators

**AI Metrics**
- Overall AI detection trends
- High-risk submission alerts
- Class-by-class comparison

## Demo Account Limitations

If you're using a demo account, be aware of these limitations:

| Feature | Demo Limit |
|---------|------------|
| File Uploads | Maximum 3 files |
| Storage | Standard quota applies |
| AI Lab | May be locked |

To remove limitations, upgrade to a full account.

## Quick Actions

### From the Dashboard, you can:

1. **Create a New Class**
   - Click **+ New Class** button
   - Enter class details
   - Start adding students

2. **View Recent Submissions**
   - Click any submission card
   - Opens grading interface directly

3. **Check Notifications**
   - Click the bell icon
   - View new enrollments and submissions
